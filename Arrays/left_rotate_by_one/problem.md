# Left Rotate Array by One

## Problem Statement
Given an array of integers, rotate the array to the left by one position.

In a left rotation, the first element of the array moves to the end, and all other elements shift one position to the left.

## Example 1
Input:
nums = [1,2,3,4,5]

Output:
[2,3,4,5,1]

## Example 2
Input:
nums = [10,20,30]

Output:
[20,30,10]

## Constraints
- 1 ≤ nums.length ≤ 10⁵
- -10⁹ ≤ nums[i] ≤ 10⁹

## Notes
- The rotation should be done in-place if possible.
- No extra array should be used for optimal solution.
