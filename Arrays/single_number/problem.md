# Single Number

## Problem Statement
Given a non-empty array of integers `nums`, every element appears **twice** except for one. Find that single one.

You must implement a solution with a linear runtime complexity and use only constant extra space.

## Example 1
Input:
nums = [2,2,1]

Output:
1

---

## Example 2
Input:
nums = [4,1,2,1,2]

Output:
4

---

## Example 3
Input:
nums = [1]

Output:
1

---

## Constraints
- 1 ≤ nums.length ≤ 3 × 10⁴
- -3 × 10⁴ ≤ nums[i] ≤ 3 × 10⁴
- Each element appears twice except for one element

## Notes
- Try solving without using extra data structures.
- Can you do it in O(n) time and O(1) space?
